# Neopets Activity Tracker

A browser extension that helps track Neopets activities, cooldowns, and daily resets,
and notifies you when activities are available again.

## What this does
- Tracks activity availability (daily resets, cooldown timers)
- Provides quick links to activities
- Sends optional notifications
- Does NOT automate gameplay or perform actions

## Disclaimer
This project is not affiliated with Neopets.
It does not automate gameplay or bypass site mechanics.
All actions are performed manually by the user.
